# API Endpoints Map (Week 11 Planning)

| Module | Method | Endpoint | Purpose | Timing |
|---|---|---|---|---|
| Auth | POST | /api/auth/login | Login user and return token/user data | Week 12 |
| Auth | GET | /api/auth/me | Get current logged-in user | Week 12 |
| Users | GET | /api/users | List staff accounts | Week 12-13 |
| Users | POST | /api/users | Create staff account | Week 12-13 |
| Users | PUT | /api/users/:id | Update staff account | Week 12-13 |
| Users | PATCH | /api/users/:id/deactivate | Deactivate staff account | Week 12-13 |
| Categories | GET | /api/categories | List categories | Week 13 |
| Products | GET | /api/products | List/search/filter products | Week 13 |
| Stock | POST | /api/stock/in | Record stock-in | Week 13 |
| Stock | POST | /api/stock/out | Record stock-out | Week 13 |
| Reports | GET | /api/reports/low-stock | View low-stock report | Week 14 |
